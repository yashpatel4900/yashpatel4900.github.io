A B O U T

Hey! Nice to see you here.

I am in the field of information and communication technology and completed 2 years of the course.

Information and communication Technology

Information and communication Technology refers to technologies that provide access to information through telecommunications. It is similar to Information Technology (IT), but focuses primarily on communication technologies. This includes the Internet, wireless networks, cell phones, and other communication mediums.

Things I am good at:

Python

I've always thought Python was easier to learn, so it's more accessible to most beginning coders. That was actually the thing that convinced me to learn it in the first place. Secondly, it has much better readability than C++ or Java. I also like that when I run into a problem, there is usually more than one tried and true way to resolve it.

I have used python for Web Scrapping of stock prices of Bombay Stock Exchange Companies using Beautifulsoup library and also built a game Space Invader.

Web Development

A web developer designs, develops, enhances, tests and deploys web applications with an end goal of creating engaging and user-friendly site layout and function.

This rough framework outlines my process of developing website:

  1. Gather and define requirements
  2. Maintaining website
  3. Troubleshoots and fixes bugs
  4. Follow best practices and collaborate with other teams

I became fascinated the ability a web developer has to be imaginative, artistic and technical that benefits and inspires others.

Working on my own projects is usually where I push the boundaries.

YOU DONT HAVE TO BE GREAT TO START, BUT YOU HAVE TO START TO BE GREAT.
